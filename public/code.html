The basic unit of defining and calling Clojure code is the function. Of course, functions are values. Top-level functions are stored in references, so they can be atomically swapped at runtime.

Mycroft shows the documentation string for a function, plus the source code. So, by browsing the mycroft.* namespaces, you can see all of Mycroft's implementation bits from within Mycroft.
