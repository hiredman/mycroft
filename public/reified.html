In order to have a generic inspector, you need to have a top-level notion of "show me everything." Clojure provides reified namespaces, which unify with Java packages and provide a top-level entry point. Mycroft's web interface lets you:

  • browse all namespaces
  • browse vars in a namespace

Vars, in turn, point to the all the pieces of your program: values, references, and code.
